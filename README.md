# Google Gemini Chatbot using Flowise

## Overview

This project is a simple AI chatbot built using Flowise and Google's Gemini 2.5 Flash model.

The chatbot uses:
- Prompt Template
- LLM Chain
- Google Gemini API

## Features

- Conversational AI chatbot
- Prompt engineering using templates
- Gemini API integration
- Low-code development using Flowise

## Workflow

User Input → Prompt Template → Gemini 2.5 Flash → LLM Chain → Response

## Tech Stack

- Flowise
- Google Gemini API
- Gemini 2.5 Flash
- LLM Chain
